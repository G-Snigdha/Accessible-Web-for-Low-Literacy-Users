Getting Started

System Requirements

  • Google Chrome browser (version 88 or later)
  • Active internet connection
  • Permissions for accessing selected text and storage

Installation Steps

  1. Download the extension package
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode"
  4. Click "Load unpacked" and select the extension directory
  5. Verify the extension icon appears in your toolbar

Features

Text Simplification

The text simplification feature uses Chrome's Built-in AI to convert complex text into easier-to-understand language.

How to use:

  1. Select text on any webpage
  2. Click the extension icon
  3. Click "Simplify Text" or use Ctrl/Cmd + Shift + S
  4. Review the simplified version in the overlay
  5. Click "Apply" to replace the original text

Translation

Translate selected text into multiple languages while maintaining context and meaning.

How to use:

  1. Select text to translate
  2. Click the extension icon
  3. Click "Translate" or use Ctrl/Cmd + Shift + T
  4. Choose your target language
  5. Review and apply the translation

Proofreading

AI-powered proofreading to improve text quality and correctness.

How to use:

  1. Select text to proofread
  2. Click the extension icon
  3. Click "Proofread" or use Ctrl/Cmd + Shift + P
  4. Review suggested corrections
  5. Apply the changes

Text-to-Speech

Convert text to natural-sounding speech with multi-language support.

How to use:

  1. Select text to read
  2. Click the extension icon
  3. Click "Read Aloud" or use Ctrl/Cmd + Shift + R
  4. Use playback controls to pause/resume

Troubleshooting

Extension Not Working

  • Verify Chrome is up to date
  • Check if the extension is enabled
  • Try reloading the webpage
  • Reinstall the extension if necessary

Features Not Responding

  • Ensure text is properly selected
  • Check internet connectivity
  • Verify permissions are granted
  • Clear browser cache and reload
